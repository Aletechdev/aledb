ALE Analytics

This web platform has been built to support the report generation and consolidation of adaptive mutations from Adaptive Laboratory Evolution (ALE) experiments. Please see the following work for a full description of the platform: thesis report

Mutation data generation

The mutation data presented on this platform was generated using the breseq resquencing pipeline.

Deatherage, D.E., Barrick, J.E. (2014) Identification of mutations in laboratory-evolved microbes from next-generation sequencing data using breseq. Methods Mol. Biol.
